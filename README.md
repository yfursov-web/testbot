# 🍕 Тренажер по решению конфликтов для Управляющих (Telegram Bot)

Этот бот — диалоговый ИИ-тренажер, разработанный для отработки управленческих навыков в конфликтных ситуациях на базе стандартов и ценностей «Додо Пиццы». 

Нейроагент выступает в роли конфликтного сотрудника, а пользователь (Управляющий) должен решить проблему, опираясь на алгоритм решения конфликтов, эмпатии и соблюдения стандартов. В конце каждого диалога бот выступает в роли асессора и оценивает действия Управляющего.
